# API тестирование: PUT, PATCH, DELETE

## PUT-запрос

- URL: https://reqres.in/api/users/2
- Метод: PUT
- Код ответа: 200 OK
- Ответ:
 
```json
{
    "name": "Artyom",
    "job": "QA Engineer",
    "updatedAt": "2025-07-18T09:02:45.698Z"
}
```

## PATCH-запрос

- URL: https://reqres.in/api/users/2
- Метод: PATCH
- Код ответа: 200 OK
- Ответ:
 
```json
{
    "name": "Artyom",
    "job": "Middle QA soon",
    "updatedAt": "2025-07-18T09:05:05.459Z"
}
```

## DELETE-запрос

- URL: https://reqres.in/api/users/2
- Метод: DELETE

```json

1
```
